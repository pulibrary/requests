# PUL Requests Engine

[![Build Status](https://travis-ci.org/pulibrary/requests.svg?branch=master)](https://travis-ci.org/pulibrary/requests)

## Install

* Add ```gem 'requests', :git 'git@github.com:pulibrary/requests.git'```
* ```bundle install```
* ```rails generate requests:install```
